# Community Event Website

Responsive Webseite für eine Gemeinschaftsveranstaltung (Younited) – Programm, Anmeldung und Infoseiten.

## Features

- Landingpage mit Veranstaltungsdetails und Programm
- Einbindung von Flyern und Vereinslogo
- PDF-Bereitstellung für Veranstaltungsunterlagen
- Deployed auf Netlify

## Tech Stack

HTML · CSS · JavaScript · Netlify

## Deployment

```bash
netlify deploy --prod
```
